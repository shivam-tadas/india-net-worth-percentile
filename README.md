# India Net Worth Distribution Viewer

A sleek, interactive Vite + React application that visualizes the distribution of net worth percentiles across India. Built with modern, glassmorphism design principles, it boasts dynamic interactions, engaging micro-animations, and a responsive Area Chart designed to help users evaluate how different net worth levels relate to overall percentiles.

## Features

- 🇮🇳 **India-Centric Data**: Realistically models India's net worth landscape in INR (₹).
- 🌓 **Light/Dark Mode**: A beautiful, easily toggleable theme switch providing premium aesthetics in both light and dark settings.
- 📈 **Interactive Graph**: Powered by Recharts – hover over the net worth curve to unveil highly detailed, descriptive tooltips matching each percentile!
- 🔮 **Glassmorphism UI**: Uses visually striking translucent components and radial gradients for an ultra-premium feel. 

## Technology Stack

- **React Framework**: Vite
- **Libraries used**: `recharts` for the interactive Area Chart mapping.
- **Icons**: `lucide-react` for premium scalable SVG icons.
- **Styling**: Native CSS implementation optimized via root variables for theming.

## Getting Started

1. Clone or download the repository.
2. Initialize and install necessary packages:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open the link provided in the CLI output. (Typically `http://localhost:5173/` or `http://localhost:5174/`)
